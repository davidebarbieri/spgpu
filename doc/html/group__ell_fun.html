Replaces the values at coordinate (aI[i], aJ[i]) inside A with the value aVal[i] * alpha, with A stored in ELLpack Format on GPU. It assumes that indices from the same row inside rP are sorted by ascending order. Values are single precision floating point complex numbers.

Parameters:
handleThe spgpu handle used to call this routine
alphaThe alpha scalar
cMThe ELL non zero values allocation pointer
rPThe ELL column indices allocation pointer
cMPitchthe pitch (in number of elements) of the allocation containing the matrix non zero values
rPPitchthe pitch (in number of elements) of the allocation containing the matrix non zero column indices
rSthe array containing the row sized (in non zero elements)
nnzthe number of triples (aI, aJ, aVal) to process
aIThe row coordinates vector
aJThe column coordinates vector
aValThe values vector
baseIndexthe ELL format base index used (i.e. 0 for C, 1 for Fortran).

Computes single precision complex z = alpha*A*x + beta*y, with A stored in ELLpack Format on GPU.

Parameters:
handleThe spgpu handle used to call this routine
zThe output vector of the routine. z could be y, but not y + k (i.e. an overlapping area over y, but starting from a base index different from y).
yThe y input vector
alphaThe alpha scalar
cMThe ELL non zero values allocation pointer
rPThe ELL column indices allocation pointer
cMPitchthe pitch (in number of elements) of the allocation containing the matrix non zero values
rPPitchthe pitch (in number of elements) of the allocation containing the matrix non zero column indices
rSthe array containing the row sized (in non zero elements)
rIdx(optional) An array containing the row index per every row (i.e. the reorder array) of the Ell matrix. Pass NULL if you don't use a reorder array (i.e. the k-th row is stored in the k-th position in the ELL format).
avgNnzPerRow(optional) Average number of non zeroes per row. Pass 0 if you don't have such information.
maxNnzPerRowMaximum number of non zeroes per row.
rowsthe rows count
xthe x vector
betathe beta scalar
baseIndexthe ELL format base index used (i.e. 0 for C, 1 for Fortran).
