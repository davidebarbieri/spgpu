Computes double precision z = alpha*A*x + beta*y, with A stored in Diagonal Format on GPU.

Parameters:
handleThe spgpu handle used to call this routine
zThe output vector of the routine. z could be y, but not y + k (i.e. an overlapping area over y, but starting from a base index different from y).
yThe y input vector
alphaThe alpha scalar
dMThe DIA non zero values allocation pointer (stored column-major)
offsetsThe DIA diagonals offsets vector
dMPitchthe pitch (in number of elements) of the allocation containing the matrix non zero values
rowsthe rows count
colsthe columns count
diagsthe diagonals count
xthe x vector
betathe beta scalar

Computes double precision z = alpha*A*x + beta*y, with A stored in Hacked Diagonal Format on GPU.

Parameters:
handleThe spgpu handle used to call this routine
zThe output vector of the routine. z could be y, but not y + k (i.e. an overlapping area over y, but starting from a base index different from y).
yThe y input vector
alphaThe alpha scalar
dMThe stacked HDIA non zero values allocation pointer
offsetsThe stacked HDIA diagonals offsets vector
hackSizeThe constant size of every hack (must be a multiple of 32)
hackOffsetsthe array of base index offset for every hack of HDIA offsets vector, plus a last value equal to the size of the offsets vector
rowsthe rows count

xthe x vector
