spgpuStatus_t spgpuCreate ( spgpuHandle_t *  pHandle,
int  device 
)

Create a spgpu context for a GPU device. Every call to spgpu routines using this handle will execute on the same GPU. This is re-entrant, so it will work if used by multiple host threads.

Parameters:
pHandleoutputs the handle
deviceid of the device to be used by this context
