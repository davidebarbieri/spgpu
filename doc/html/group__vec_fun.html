const __device int *  xIndices,
int  xBaseIndex,

)

Single precision complex scatter from sparse(x) to y. Computes the single precision complex scatter from xValues to y (using xIndices). The scattered element will be, for i in [0,xNnz), y[xIndices[i]] = beta*y[xIndices[i]] + xValues[i] (to be noted that y values will be multiplied with beta just for scattered values).

Parameters:
handlethe spgpu handle used to call this routine
ythe destination vector (to which the elements will be scattered)
xNnzthe number of elements to scatter
xValuesthe source array from which the values will be read
xIndicesthe array of indices for the elements to be scattered
xBaseIndexthe base index used in xIndices (i.e. 0 for C, 1 for Fortran).
betathe beta value
